## 产品简介

**FileSender（文件传输）** 是一款简洁高效的跨设备文件传输应用。通过扫描二维码即可快速将照片、视频从手机传输到电脑或其他设备，无需数据线、无需登录账号。

**核心价值**：
- 📱 **极简操作**：选择文件 → 扫码 → 完成传输
- 🚀 **智能压缩**：自动压缩照片和视频，节省流量和时间
- 🔒 **隐私安全**：点对点传输，文件不经过第三方服务器
- 💨 **快速高效**：局域网传输速度快，支持批量传输

## 特色亮点

### 🚀 极简设计

- **三步完成**：选择 → 压缩 → 扫码传输
- **清晰界面**：每个步骤都有明确的视觉引导
- **流畅体验**：页面切换自然流畅

### 🔒 隐私保护

- **点对点传输**：文件直接从发送端传到接收端
- **不经过云端**：无需上传到第三方服务器
- **临时存储**：压缩后的临时文件自动清理
- **局域网优先**：优先使用局域网传输，更快更安全

### 💡 智能优化

- **自动压缩**：根据文件类型自动选择最佳压缩策略
- **批量处理**：支持一次性处理多个文件
- **错误处理**：网络异常时自动重试

### 📱 跨平台支持

- **iOS 应用**：原生 Flutter 开发，性能优秀
- **配合接收端**：需要配合接收端应用或网页使用
- **局域网传输**：同一 WiFi 下传输速度最快

## 使用场景

### 场景一：手机照片传电脑
**问题**：拍了很多照片想传到电脑上编辑，用数据线太麻烦

**解决方案**：
1. 打开 FileSender，选择要传输的照片
2. 应用自动压缩照片
3. 在电脑上打开接收端，扫描二维码
4. 照片自动传输到电脑

---

### 场景二：视频快速分享
**问题**：录制的视频文件太大，通过社交软件发送会被压缩画质

**解决方案**：
1. 选择视频文件
2. 智能压缩到合适大小
3. 扫码传输到接收设备
4. 保持较好的视频质量

---

### 场景三：批量文件传输
**问题**：需要一次性传输多个文件到另一台设备

**解决方案**：
1. 批量选择多个照片和视频
2. 一键压缩所有文件
3. 扫码后自动依次传输
4. 实时查看传输进度

## 使用流程

### 第一步：选择文件
- 点击"选择文件"按钮
- 从相册中选择要传输的照片或视频
- 支持多选

### 第二步：压缩优化
- 应用自动压缩选中的文件
- 显示压缩进度
- 压缩完成后进入扫码页面

### 第三步：扫码传输
- 在接收端设备上打开接收程序
- 使用 FileSender 扫描接收端显示的二维码
- 自动开始传输

### 第四步：完成
- 查看传输结果
- 可选择继续传输更多文件
- 或返回首页

## 常见问题

### Q1: 需要两台设备都安装应用吗？
**A**: 发送端需要安装 FileSender 应用，接收端可以是应用或网页程序。

### Q2: 传输速度快吗？
**A**: 在同一 WiFi 局域网下传输速度很快，通常可达到 10-50 MB/s。使用移动网络会受网速限制。

### Q3: 文件会保存在云端吗？
**A**: 不会。FileSender 采用点对点传输，文件直接从发送端传到接收端，不经过任何云服务器。

### Q4: 压缩会损失画质吗？
**A**: 会有轻微的画质损失，但在可接受范围内。我们使用智能压缩算法，在文件大小和画质之间取得平衡。

### Q5: 支持传输哪些文件类型？
**A**: 目前支持照片（JPEG、PNG、HEIC 等）和视频（MP4、MOV 等）。

### Q6: 传输失败怎么办？
**A**: 请检查网络连接，确保两台设备在同一网络下。如果问题持续，可��尝试重新扫码连接。
