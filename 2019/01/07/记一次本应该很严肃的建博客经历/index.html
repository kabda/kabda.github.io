记一次本应该很严肃的建博客经历

发表于 2019-01-07 | 分类于 人生杂谈

毕竟懒癌晚期，博客搭了快一周了，还是没有动力去真的写点东西，也没有什么思路，要不，就讲讲我为什么这次“又”要搭博客吧，至于为什么要用“又”，且听我慢慢道来。

就在大概5天前，一个朋友在微信上找我
T：你有个人博客么？
W：没有啊，怎么了？
T：我X，你都干了5年开发了，竟然没有博客？
W：有什么直接关系么？
T：我X，你平时不做总结的么？不做技术调研的么？不记录的么……此处省略一万字……
W：哦，那我去搭一个吧。

好吧，被鄙视了怎么能忍，说干就干，毕竟现在是一个强调“执行力”的时代，Google了一个教程，在本地捣鼓了一个钟头，正准备发布的时候，神奇的事情出现了。

我的 Github Pages里，静静地躺着一个两年半未更新的博客。

不管它，消灭证据先，直接发布新的博客，毕竟正面形象还是很重要的。

T：不对啊，哥们，我看你的 Github 里有一个博客啊，还是16年搭建的，还 TMD 只有一个 Hello World。
W：怎么可能呢，你看错了吧，刷新一下试试！
T：我X，你干了啥……

不过，既然都已经是第二次搭博客了，索性就学着写点东西吧，也不强调说一定要是技术性的（关键是本人技术水平实在是渣），职业上的，生活上的，感情上的，能写什么就写什么吧，不被查水表就行。刚好2019年，我给自己定的目标是“极致执行力”，那就以此为起点，给自己立个 flag，每周至少写一篇。嗯，就这么干！
